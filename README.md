#### Register
Register to the system, email must be unique.
- [ ] POST /register

Body to send
```js
{
    firstName: "daniel",
    lastName: "philosoph",
    email: "danielph9@gmail.com",
    password: "123456789"
}
```

Response
```js
{
    user: {
        firstName: "daniel",
        lastName: "philosoph",
        email: "danielph9@gmail.com",
        _id: "6212133cc7ff924493558d23",
        entries: []
    },
    registered: true
}
```

Request for all users. Authorization is needed!
- [ ] GET /user/users
